## 它到底是个啥

BandiView 是 Bandisoft 出的 Windows 图像查看器,你可以把它理解成老牌看图软件 Honeyview 的"全面升级版"。Honeyview 当年口碑就不错,可惜停在 XP 到 Win11 那个年代,不再更新了。BandiView 接过了接力棒,把前代所有好用的功能都继承了过来,然后又加了一堆专业向的东西。

版本号我写这篇的时候是 v7.27,2026 年 5 月 23 日更新的。只支持 Windows 10 和 11 的 64 位版本——这点得先说清楚,Mac 用户就别往下看了,会失望。

它的核心卖点其实就一句话:**什么格式都能看,看得还特别快。**

---

## 格式支持:一个看图软件的"胃口"

我先列一下它胃口有多大。

常见格式 BMP、JPG、GIF、PNG、PSD、WebP、TGA、TIFF 这些就不说了,基本是看图软件的入门门槛。它还能啃相机原档——DNG、CR2、CRW、NEF、NRW、ORF、RW2、PEF、SR2、RAF,主流相机的 RAW 基本通吃。

专业版再往上加一截:AVIF、JXL、EXR、QOI、ICO、SVG、最新的 RAW 格式、Animated AVIF,还有 iPhone 那套 HEIC / HEIF / HIF,以及 Clip Studio Paint 的 CLIP 工程文件。

这意味着什么呢?意味着你拿 iPhone 拍的Live Photo、相机直出的 RAW、画师发来的 CLIP 工程、网上下的 AVIF 动图、Stable Diffusion 跑出来的 WebP——双击,全都能直接看。不用先开 Photoshop,不用先转格式,不用先解压。

对,它还支持压缩包直读。ZIP、RAR 里的图片,不解压就能翻着看。漫画党看到这里应该已经在点头了。

## HDR 输出:你可能没用过,但用了就回不去

BandiView 支持 HDR(高动态范围)图像输出。如果你有一块 HDR 屏幕,它能让你看到更宽的亮度范围和更深的色彩层次——亮部不至于死白,暗部不至于糊成一团。

支持 HDR 输出的格式包括 Ultra HDR(JPEG)、HEIC、AVIF、EXR、JXR、PSD、DDS、JXL、Radiance HDR。

我知道很多人会说"我这显示器又不 HDR,这功能跟我有啥关系"。没关系,这功能是给那些有 HDR 屏的摄影师和设计师准备的——他们拍 HDR 照片、做 HDR 渲染,以前要预览效果得开一堆专业软件,现在双击就行。对没 HDR 屏的人来说,这个功能不影响日常使用,就当它不存在。

## 漫画模式:垂直一滚到底

这是我个人最喜欢的一个功能。

BandiView 有个"网络漫画视图模式",可以把多张图片当成一条垂直流来显示,你往下滚,图就连续地铺过来。看条漫、看长图特别舒服,不用一张张点下一页,符合人在手机上养成的滚动阅读习惯。

配合压缩包直读,体验是这样的:下载一个漫画压缩包,双击,BandiView 直接打开里面的图,切到漫画模式,往下滚,一路看下去。整个过程你不需要解压,不需要翻页,不需要任何额外操作。
